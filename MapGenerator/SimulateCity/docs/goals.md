# CivicOverreach — Goals

## Primary Goals

1. Create believable abandoned civic structures.
2. Ensure bridges exist in risky but narratively interesting locations.
3. Scatter abandoned buildings that players can repurpose.
4. Make the world feel previously inhabited and overextended.
5. Support salvage, repair, and reuse gameplay loops.

---

## Secondary Goals

- Provide heuristic data explaining why ruins exist.
- Enable future lore, AI, and civilization analysis stages.
- Allow optional validation via external engines.

---

## Explicit Non-Goals

- Creating balanced or optimal layouts.
- Ensuring every area is playable or reachable.
- Simulating real economics or population.
- Producing reproducible outputs.
- Requiring external engines to run the pipeline.

CivicOverreach is allowed to be messy, opinionated, and imperfect.
